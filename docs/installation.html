Installation

BRAPHIN requires Python 3.9 or later. It is tested on Linux, macOS, and Windows.

Requirements

The core library depends only on well-established scientific Python packages:

PackageMinimum versionPurpose
nibabel≥ 5.0NIfTI file loading and resampling
numpy≥ 1.24Array operations throughout the pipeline
scipy≥ 1.10Cross-correlation computation
networkx≥ 3.0Graph construction and export

Core Install

Install the core fMRI library (no EEG or GNN dependencies):

pip install -r requirements.txt

Or, if you have a local checkout with pyproject.toml:

pip install .

Optional Extras

EEG support

Adds MNE-Python and supporting packages for the original EEGraph EEG pipeline:

pip install -r requirements-eeg.txt
# — or via pyproject extras —
pip install ".[eeg]"
PackageVersionPurpose
mne≥ 1.5EEG/MEG data I/O and signal processing
pandas≥ 2.0Tabular data handling in EEG pipeline
antropy≥ 0.1.6Entropy measures (EEG connectivity)
plotly≥ 5.0Interactive graph visualisation
scot≥ 0.19Spectral connectivity (EEG)

GNN support

Adds PyTorch and PyTorch Geometric for graph neural network classification:

pip install -r requirements-gnn.txt
# — or via pyproject extras —
pip install ".[gnn]"
PackageVersionPurpose
torch≥ 2.0PyTorch deep learning framework
torch-geometric≥ 2.4GNN layers and data loaders
scikit-learn≥ 1.3Train/test splits, metrics
pandas≥ 2.0Tabular data handling
PyTorch install note: torch-geometric has specific version requirements per CUDA version. Follow the official PyG installation guide for your hardware before running pip install ".[gnn]".

Install from Source

Clone the repository and install in editable mode so changes to the source are immediately reflected:

git clone https://github.com/<your-org>/braphin.git
cd braphin
pip install -e .

To include all optional extras at once:

pip install -e ".[eeg,gnn,dev]"

Verifying Your Install

A successful install should import without errors and show the version:

import braphin
print(braphin.__version__)   # 1.0.0

# Confirm bundled atlases are available
from braphin.atlas import list_supported_atlases
print(list_supported_atlases())
# ['aal', 'schaefer_100', 'schaefer_200', 'schaefer_400']

Developer Install

To run the test suite and contribute to BRAPHIN:

# 1. Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate

# 2. Install in editable mode with dev dependencies
pip install -e ".[dev]"
# — or —
pip install -r requirements-dev.txt && pip install -e .

# 3. Run the test suite
pytest                                   # 387 tests
pytest --cov=braphin --cov-report=term-missing   # with coverage
Tip: All 387 tests use synthetic data — no NIfTI files or internet access required.
